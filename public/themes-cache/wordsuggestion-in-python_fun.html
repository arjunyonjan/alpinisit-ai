🚀 WordSuggestion-in-Python

Word Suggestion with N-Grams and Cosine Similarity ✨

💡 Implemented in Python from Scratch — No External ML Libraries!

📖 Project Overview

This project demonstrates a fundamental approach to building a word suggestion or autocomplete system without relying on external machine learning libraries. It leverages character-level n-grams and cosine similarity to find the closest matching words from a dictionary. 🎯

⚙️ Algorithm Overview (12 Steps!)

1

📂 Read All Words from File

word_list = read_all_words()

Opens dictionary.txt and reads every line into a Python list. Each line is treated as a single word entry. The file may contain tens of thousands of words.

2

🎲 Sample and Sort 1000 Words

sorted_sampled_words = sample_1000_words(word_list)
  • Filters out any word shorter than MINIMUM_LENGTH = 3 characters (e.g., "a", "to" are discarded).
  • Randomly samples 1000 words from the filtered list using random.sample().
  • Sorts the 1000 words alphabetically.

✨ Example: From 80,000+ words in the dictionary, 1000 are selected, e.g., ['abandon', 'ability', 'above', ...]

3

🔤 Build the N-Gram Dictionary

word_ngram_dictionary = build_ngram_dictionary(sorted_sampled_words)

For every sampled word, generates all character-level n-grams of sizes NGRAM_SIZES = [3, 4, 5]. The result is a dictionary mapping each word to its list of n-grams.

🌟 Example for "blue":

  • 3-grams → ['blu', 'lue']
  • 4-grams → ['blue']
  • 5-grams → [] (word is too short)
  • Combined → ['blu', 'lue', 'blue']
4

📊 Build N-Gram Frequencies

ngram_frequencies = build_ngram_frequencies(word_ngram_dictionary)

Iterates over every n-gram from every word and counts how many times each unique n-gram appears across the entire sampled dictionary. Creates the global n-gram frequency table.

💡 Example: If 'ing' appears in 200 of the 1000 words, then ngram_frequencies['ing'] = 200.

5

📐 Calculate Vector Dimensions

dimension = len(ngram_frequencies.keys())

The total number of unique n-grams across all sampled words defines the dimensionality of the vector space. Every word will be represented as a vector of this length.

🎯 Example: If there are 4500 unique n-grams, every word vector has 4500 dimensions!

6

📋 Sort N-Gram Positions

ngram_positions = list(ngram_frequencies.keys())
ngram_positions.sort()

Extracts all unique n-grams and sorts them alphabetically. This sorted list determines the fixed ordering of dimensions in every word vector.

7

🗺️ Build the Position Dictionary

position_dictionary = build_position_dictionary(ngram_positions)

Maps each unique n-gram to a fixed integer index (its position in the vector). Used during vectorization to know which index a given n-gram corresponds to.

8

🧮 Build All Word Vectors

ngram_vectors = build_word_vectors(word_ngram_dictionary, position_dictionary, ngram_frequencies)

For each word, creates a zero-filled array of dimension length, then places the global n-gram frequency at the corresponding index position for each n-gram the word contains.

9

⌨️ Read the Input Word and Vectorize It

input_word = input("\nGive a word : ")
input_vector = vector_of(input_word, ...)

Prompts the user for a word. The input word is converted into a vector using the same position dictionary and n-gram frequencies.

10

📏 Calculate Cosine Similarity

angles_with_input = calculate_angles_with(input_vector, ngram_vectors)

Computes cosine similarity for every word using the formula: cos(θ) = (v · w) / (||v|| × ||w||)

✨ Example result: { 'blue': 0.93, 'bluebell': 0.81, 'bluet': 1.0, ... }

11

🏆 Rank Words by Similarity Score

ranked_words = ranked_words_of(angles_with_input)

Sorts all words by cosine similarity score in descending order and selects the top SUGGESTION_SIZE = 10 results.

12

🎯 Filter Zero-Scored Words & Print Results

final_suggestions = filter_zero_scored_words(ranked_words)

Removes words with similarity score of 0.0 and prints the final suggestions!

Give a word : bluet

Final Suggestions :

>> bluet: 1.0

>> blue: 0.93

>> bluebell: 0.81

>> bluets: 0.79

>> blur: 0.45
